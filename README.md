# MinMax

Ett odds- och sannolikhetsverktyg för sportsbetting, byggt som en enda HTML-fil. Beräknar förväntade odds och vinstchansen per match med hjälp av binomialfördelning.

**Live:** https://vg1414.github.io/MinMax/

## Funktioner

- **Odds per match** – Mata in odds och beräkna sannolikheter
- **Binomialfördelning** – Statistisk uträkning av vinstchans
- **MinMax-analys** – Hitta optimala insatser

## Teknik

- Vanilla HTML/CSS/JavaScript – inga ramverk
- Matematiska funktioner: binomialkoefficienter, sannolikhetsfördelningar
- Google Fonts (Bebas Neue, DM Sans)
- Mörkt tema med gula accenter
